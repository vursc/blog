(WORK-IN-PROGRESS)

我听说赛后复盘对水平的提高是很重要的，但是一直没有这样的习惯，所以就把总结写在这里，算是激励自己坚持下去。

东南赛

虽然我是新高二的考生，但是东南赛还是无耻地报了高一组。高一组的题整体不难，没有考场上不可做的题目。Day 1 的 P1 直接写出 ana_nan​ 通项求和即可。P2 首先注意到分奇偶可做 n=2n = 2n=2，然后 n=3n = 3n=3 时再将奇数模 444 分组即可，于是容易想到模 2n2^n2n。实际的构造可以归纳，也可以直接将模 22r2^{2r}22r 余 k⋅2r+1k \cdot 2^r + 1k⋅2r+1 的数单独分组，然后余下的数为一组，得到 2r+12^r + 12r+1 组。P3 我没做出来，目前已知的大部分做法需要反演。其实我考场上考虑过反演，但是因为对反演不熟悉，没有细想；当时是立刻看出 LLL 在 ⊙BIC\odot BIC⊙BIC 上，但是一直不会刻画 ⊙BKJC\odot BKJC⊙BKJC。P4 直接刻画集族的形式即可，不过用小 nnn 不容易找到规律。我最初没注意到可能有多个元素在所有集合中，最大值 2n2n2n 的证明快写完才发现错误，直接换答题卡（笑）。

然后是 Day 2。P1 注意到 AB⊥OD⊥PQAB \perp OD \perp PQAB⊥OD⊥PQ 且条件都在边上，直接设边长算 tan⁡∠PBA\tan \angle PBAtan∠PBA 和 tan⁡∠QAB\tan \angle QABtan∠QAB 然后用和角公式即可，不过几何做法也并不难。P2 直接归纳，随便乱放。P3 枚举题，猜到答案直接硬讨论。P4 是经典的考虑最大素幂的手法，默写一遍就成（利用 (aix−bi,ajx−bj)∣(aibj−ajbi)(a_i x - b_i, a_j x - b_j) \mid (a_i b_j - a_j b_i)(ai​x−bi​,aj​x−bj​)∣(ai​bj​−aj​bi​) 给出最大公因数的上界）。

其实第一天做 P1 的时候我拆出三个求和式，没想到相邻项可以抵消，试图分开算求和，浪费了大概二十分钟。有几个同学也是一样，不过好在题不难，而且放在一号位，怎么都能做出来，只是花费时间多少的问题。

东南赛的题目质量确实不高。D1P1 根本就是一试题，D2P3 也只是麻烦的讨论（甚至讲的答案还是错的……我表示无法理解）。

北大夏令营

然后就是北大夏令营。我是考试前一天上午坐高铁去的北京，坐完高铁之后状态不太好，下午报到之后就回酒店休息了。晚上因为床太软睡得很不好，做了几个噩梦，把自己吓醒了几次。Day 1 的 P1 反证，假设 cos⁡(pπ/q)=1/n\cos (p \pi / q) = 1/\sqrt{n}cos(pπ/q)=1/n​，于是 cos⁡(2pπ/q)=2/n−1\cos (2p \pi / q) = 2/n - 1cos(2pπ/q)=2/n−1。由 qqq 阶切比雪夫多项式 fqf_qfq​ 首项系数为 2q2^q2q 知 1=fq(2/n−1)∉Z1 = f_q(2/n - 1) \not\in \Z1=fq​(2/n−1)∈Z 矛盾。P2 考虑在每一位上的进位次数即可，不好叙述，我应该是过程写得不好丢了分。P3 没做出来，只注意到 BL,CF,DQ,JRBL, CF, DQ, JRBL,CF,DQ,JR 中任两组的四点共圆；有同学算 ⊙ABC\odot ABC⊙ABC 和 ⊙NFRQL\odot NFRQL⊙NFRQL 的幂差做出来了，但是不太好做；据说用根轴是简单的，但是我还没仔细想。P4 只是猜对了答案，同学跟我说了大致的思路，稍后再整理。

第一天的考试，我只做了 P1 和 P2 两个题，而且前者是在考试快结束时才做出来的。在大概一个半小时的时候感觉非常困，睡了半个小时再继续把 P2 做完。时间可能是足够再做一个 P4 的，但是那样 P1 就不一定想得出来，而且每个黑格都在 2×22 \times 22×2 内的条件并不好用。

Day 2 的难度较低。P1 先齐次化，然后只需 xn,…,x1x_n, \dots, x_1xn​,…,x1​ 的系数列优超 0,…,00, \dots, 00,…,0 即可。P2 对 deg⁡f\deg fdegf 归纳，据说直接用 Hensel 也能做；我没来得及细想，证了一遍 Hensel 就开始乱写。P3 只需利用 M=2M = 2M=2 的条件，将方格表划分为 N×NN \times NN×N 个 2×22 \times 22×2 块，魔术师骨牌要么在一块里面（此时用一张骨牌将这一块的另两格盖上），要么横跨两块（此时用三张骨牌将这两块的余下六格盖上）。我听说本题有用 Hall 的做法，所以不知道出题人是故意放了一个简单题还是没想到前面的简单方法而认为此题较难（不过应该不会发生这种事吧）。P4 因为每列的元素都是前一列元素的平均，感觉每列的元素会逐渐靠近（不过因为可以让最大的三个相等且保持不变，最小的三个也相等且保持不变，所以元素之差不一定减小；不过这样的安排也不会产生差就是了）。考虑刻画每列数的接近程度 Δ\DeltaΔ，然后将两列之间所求的差之和放成 c⋅Δc \cdot \Deltac⋅Δ。尝试方差不行，然后想到两两距离之和，第一列（最右边的一列）放出 252525，所以 c=6c = 6c=6，尝试证明。这题我只写了考虑两两距离之和这一步，前后都没有证明，估计拿不到分。

其实没有我做不了的题，只是发挥得很不好。一方面是身体原因（两次考试都感到很困，而且没有精神做题，可能因为早餐吃得不够，或者坐出租车去学校）。另一方面，还是太紧张，很想拿奖，导致考试的时候心态不稳。其实 D2P3 和 D2P4 我完全有能力做出来，而 D1P2 和 D2P2 也可以写更好的过程。但是确实没做到，感觉很可惜。去年在清华也是因为太紧张而导致发挥得很不好，还需要多调节。

希望今年的联赛（以及潜在的新领军和金秋营）能至少正常发挥。虽然做了总结，我果然还是担心自己止步不前；即使知道这样的担心对我没有益处，也无法阻止自己。只能说，现在还没有什么办法，等时候到了再说。
